# str-textbin

A simple text pastebin service where you can upload text, view it, and share it with obfuscated URLs.

## Features

- Upload and store text
- View text with syntax highlighting
- View raw text
- Obfuscated URLs (e.g., `/ahasfn/raw/2025`)
- Optional expiry dates
- Clean, modern UI

## Usage

1. Paste or type your text
2. Optionally set an expiry date
3. Click "Create Paste"
4. Share the generated obfuscated URL

## Deployment

This project is configured for Vercel deployment. The backend API is hosted separately.
